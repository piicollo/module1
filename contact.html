Linux Bash Commands:

# Command Description
1 pwd ⏎ Path Finder
2 open . ⏎ Opens Folder
3 mkdir foldername ⏎ Create folder
4 cd foldername ⏎ Entering | Opening to created Folder
5 cd .. ⏎ Going to parent folder
5 ls ⏎ Listing files or data inside folder
6 touch filename ⏎ Create a file inside folder
7 rm filename ⏎ Delete the file inside folder
8 rm -r foldername ⏎ Delete the folder
9 mv foldername newfoldername ⏎ Rename the folder
10 mv filename newfilename ⏎ Rename a filename inside the folder
11 cat filename ⏎ Display file content to the terminal
12 vim filename ⏎ Edit the file
13 cp filename folder/filename ⏎ Copy the file from current folder to new folder
14 ls foldername ⏎ Display content inside in current folder
15 cat foldername/filename ⏎ Display content inside folder
16 ls -a ⏎ Display hidden file in current folder
